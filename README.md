# A. grafana 시작하는방법

  1. cmder 실행한다.
  2. 실행위치로 이동한다. </br>
      $> cd C:\app\grafana-7.5.17\bin
  3. 실행 명령어를 입력한다. </br>
      $> start grafana-server.exe
  4. 크롬 브라우져를 실행한다.
  5. 접속 URL 을 입력한다 </br>
      <http://localhost:3000>
  6. 계정 및 비번을 입력한다. </br>
      id : admin </br>
      pw : admin

# B. grafana 종료하는방법

  1. 실행 중인 grafana server 에서 "ctrl + c" 명령어를 입력한다.
     ->  수행중인 grafana 종료됨
